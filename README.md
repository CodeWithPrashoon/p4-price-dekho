# 🚀 P4 Price Dekho - Smart Price Companion

A high-performance **MERN Stack** application designed to help users find the best deals across India's top e-commerce platforms (Amazon, Flipkart, Croma) in real-time.

<img width="1887" height="912" alt="image" src="https://github.com/user-attachments/assets/586a96bd-0f1d-4c2e-9d51-4d7cd4189bc6" />


## ✨ Key Features
* **Real-Time Price Comparison:** Fetches live data to compare prices instantly.
* **Smart Auth System:** Secure user registration and login using **JWT** and **Bcrypt**.
* **Price Analytics:** Integrated **Recharts** to visualize price history trends.
* **User Dashboard:** Personalized Wishlist and Cart management.
* **Responsive UI:** Modern Earthy-Minimalist design built with **React**.

## 🛠️ Tech Stack
- **Frontend:** React.js, Recharts, Lucide Icons, CSS3
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Security:** JSON Web Tokens (JWT), Bcrypt.js
- **API:** SerpApi (Google Shopping Integration)

## 🚀 Installation & Setup
1. Clone the repo: `git clone https://github.com/CodeWithPrashoon/p4-price-dekho.git`
2. Install dependencies: `npm install` (in both client and server)
3. Setup `.env` with your MongoURI and API Keys.
4. Run: `npm start`
